# To-Do List Web App

A To-Do list application built using HTML, CSS, and JavaScript.

## Features
- Add tasks
- Delete tasks
- Mark tasks as completed
- Delete all tasks
- Tasks saved after refresh using localStorage

## Technologies Used
- HTML
- CSS
- JavaScript
